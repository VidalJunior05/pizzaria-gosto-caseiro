## 📁 How to download the Project 
```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/pizzaria-gosto-caseiro.git

    # Enter directory
    $ cd pizzaria-gosto-caseiro

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
